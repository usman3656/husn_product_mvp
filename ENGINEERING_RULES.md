# ENGINEERING RULES — ABSOLUTE AND NON-NEGOTIABLE

> THESE RULES ARE MANDATORY. THEY APPLY TO EVERY SINGLE LINE OF CODE AND EVERY
> SINGLE OPERATION PERFORMED IN THIS REPOSITORY, WITHOUT ANY EXCEPTION.
> EFFECTIVE: 2026-06-21.

---

## RULE 1 — HOW THE USER IS ADDRESSED

AT EVERY PROMPT AND IN EVERY SINGLE RESPONSE, THE USER IS ADDRESSED AS
"MR BAWANI" AND NOTHING ELSE. NO OTHER NAME, NICKNAME, OR FORM OF ADDRESS IS
EVER USED.

---

## RULE 2 — PARALLEL AGENTS FOR EVERY TASK

EVERY TASK, WITHOUT EXCEPTION, IS CARRIED OUT USING PARALLEL AGENTS. THE WORK
IS DECOMPOSED AND FANNED OUT ACROSS MULTIPLE AGENTS RUNNING CONCURRENTLY. NO
TASK IS COMPLETED BY A SINGLE SEQUENTIAL PASS WHEN IT CAN BE SPLIT AND RUN IN
PARALLEL.

---

## RULE 3 — RESEARCH BEFORE CODE

BEFORE ANY CODE IS WRITTEN, RESEARCH IS DONE FIRST TO CONFIRM THAT THE CHOSEN
APPROACH IS GENUINELY THE BEST WAY TO SOLVE THE PROBLEM. NO CODE IS EVER
WRITTEN ON ASSUMPTION OR GUESSWORK. THE OPTIONS ARE INVESTIGATED, COMPARED,
AND THE STRONGEST ONE IS SELECTED AND JUSTIFIED.

---

## RULE 4 — HIGHEST STANDARDS ONLY

EVERY PIECE OF CODE IS WRITTEN TO THE HIGHEST PROFESSIONAL ENGINEERING
STANDARDS. IT MUST BE CORRECT, CLEAR, MAINTAINABLE, AND CONSISTENT WITH THE
SURROUNDING CODEBASE. NO SHORTCUTS, NO BANDAID FIXES, NO SPECULATIVE CHANGES
SHIPPED ON A HUNCH.

---

## RULE 5 — MANDATORY VERIFIER AGENT

EVERY CODE CHANGE AND EVERY OPERATION THAT IS PERFORMED IS CHECKED BY A
SEPARATE VERIFIER AGENT. THE VERIFIER AGENT INSPECTS THE WORK FOR MISTAKES,
REGRESSIONS, AND ANYTHING THAT COULD CAUSE PROBLEMS LATER. NOTHING IS
CONSIDERED DONE UNTIL THE VERIFIER AGENT HAS REVIEWED IT AND CONFIRMED IT IS
SAFE AND CORRECT.

---

## SCOPE

THESE RULES GOVERN ALL FUTURE WORK IN THIS REPOSITORY. ANY NEW INSTRUCTION
THAT MR BAWANI ADDS IS APPENDED TO THIS FILE AND BECOMES EQUALLY BINDING.
